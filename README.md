Algorithmic Trading — Prediction Markets
Research and execution pipeline for trading commodity-related prediction markets (Kalshi/Polymarket), built with a co-founder. This repo covers a slice of the signal infrastructure: MCX options data collection, futures-spot basis validation, and SVI volatility surface fitting — used to build a fundamentals-grounded read on commodity price direction that feeds into prediction market positioning.
Core strategy logic and execution layer are kept private. Not all files from the full project are included here.
